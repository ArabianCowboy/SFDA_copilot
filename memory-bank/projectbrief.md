# SFDA Copilot - Project Brief

## Overview
AI-powered regulatory assistant for the Saudi Food and Drug Authority (SFDA) that provides guidance on pharmaceutical regulations and pharmacovigilance.

## Core Objectives
1. Provide accurate, AI-generated responses to regulatory and pharmacovigilance queries
2. Offer hybrid search combining semantic meaning and keyword matching
3. Maintain conversation context for complex regulatory questions
4. Simplify access to SFDA regulatory documents and requirements

## Key Features
- Chat interface with categorization (Regulatory/Pharmacovigilance/Veterinary_Medicines/Biological_Products_and_Quality_Control)
- Query expansion with pharmaceutical terminology
- Hybrid search (FAISS + TF-IDF) on regulatory documents
- OpenAI integration for generating responses
- Supabase authentication for user management
- Session management for chat history

## Target Users
- Pharmaceutical companies operating in Saudi Arabia
- Regulatory affairs professionals
- Drug safety/pharmacovigilance teams 
- Clinical trial sponsors

## Success Metrics
- Accuracy of regulatory guidance provided
- User adoption rate among target groups
- Reduction in time to find regulatory information
- User satisfaction scores
